# Student Assistant Chatbot

A simple chatbot built using Python and JSON that helps students learn different subjects and test their knowledge through quizzes.

## Features

* Multiple Subject Support
* Health Knowledge
* Programming Knowledge
* Science Knowledge
* History Knowledge
* Technology Knowledge
* Greetings Recognition
* Emotion Responses
* Quiz Mode
* Keyword-Based Search

## Requirements

* Python 3.x

## How to Run

1. Clone the repository

2. Navigate to the project folder

3. Run the chatbot

python chatbot.py

## Example Commands

* hello
* how are you
* tell me about python
* tell me about health
* start quiz
* exit quiz
